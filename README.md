
## Starter bootstrap

Стартовый шаблон для построения приложений который включает в состав:
- bootstrap 5

### Задачи gulp

1. clean - чистка директории `dist`
2. scss -  сборка css
3. html - сборка html
4. js - сборка js

### Команды

`make up` - запуск dev режима
`make build` - сборка для прода

**!! Работает с nodejs 16+**
